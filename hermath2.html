Non-Autonomous Map with Internal State Variable and External Context

This page summarizes the mathematical framework for a non-autonomous discrete map that includes:

  • Internal state variable: \(x_n\)
  • External context or control parameter: \(c_n\)
  • Time-dependent rule (non-autonomous): Update rule changes with \(n\)

1. General Mathematical Form

A non-autonomous map has an update rule that explicitly depends on time \(n\). The general form is:

\[ x_{n+1} = f_n(x_n, c_n) \]

Here:

  • \(x_n\) = internal state at step \(n\)
  • \(c_n\) = external context / environment modifying the state
  • \(f_n\) = update function that changes with time

This is the most general and correct representation.

2. Expanded Equivalent Form

We can break the update rule into components:

\[ x_{n+1} = f(x_n) + g(c_n) + h(n) \]

Where:

  • \(f(x_n)\) — internal deterministic evolution
  • \(g(c_n)\) — influence of external context
  • \(h(n)\) — explicit time-based forcing

If the internal state and context interact directly, we add a coupling term:

\[ x_{n+1} = f(x_n) + g(c_n) + k(x_n, c_n) \]

3. Fully General Non-Autonomous System

The most complete mathematical description is:

\[ x_{n+1} = F(n, x_n, c_n) \]

Here all parts — time, internal state, and external context — jointly determine the next state.

4. When Context Also Evolves (Coupled System)

Sometimes the external context \(c_n\) is itself dynamic. Then the system becomes:

\[ \begin{cases} x_{n+1} = F(n, x_n, c_n) \\ c_{n+1} = G(n, c_n) \end{cases} \]

This expresses a bidirectional relationship between internal state and external forces.

5. Example: Non-Autonomous Logistic Map

A concrete example: the logistic map with a time-varying parameter \(r_n\):

\[ x_{n+1} = r_n \, x_n (1 - x_n) \]

Where:

\[ r_n = r_0 + \alpha \sin(n) \]

In this case:

  • \(x_n\) = internal state
  • \(c_n = r_n\) = external context (varies with time)
  • \(f_n = r_n x_n(1-x_n)\)

6. Bidirectional Interaction Form (↔ Interpretation)

If the system requires a representation of feedback between state and context, the correct mathematical form is:

\[ \begin{aligned} x_{n+1} &= F(x_n, c_n) \\ c_{n+1} &= C(c_n, x_n) \end{aligned} \]

This expresses mutual influence (↔) between internal and external components.
